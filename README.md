# thearchitects
HOW TO ACCESS THE PROJECT

1. In the command line, cd into the folder thearchitects/client/SmartBite. Type the command "npm run dev". This will provide a link. Ctrl+click it 
to access the link in the browser. 
2. In a separate command line window, cd into the folder thearchitects/server. Type the command "node server.js". This will start the server and connect
to the database. 
3. Navigate to the webpage you opened in step 1. Everything should be set up and ready to go.

*Note: the only functionality we have implemented is searchProduct.
